Capture The Flag Python API Documentation¶

This page contains the API documentation for the Capture The Flag Python Starter Kit.

For more information on The AI Sandbox and the Capture The Flag Competition visit aisandbox.com. There you will find step by step guides to writing your own commander, the latest API documentation and information on how to submit your commander into the competition.
