2. Python API Documentation¶

2.1. Commander¶

class api.commander.Commander[source]¶

The base class for Commanders, that give orders to the team members. This class should be inherited from to create your own competition Commander. You must implement tick(self) in your custom Commander.

game = None¶

The GameInfo object describing this Commander’s knowledge of the current state of the game.

initialize()[source]¶

Use this function to setup your bot before the game starts. You can also set self.verbose = True to get more information about each bot visually.

issue(CommandClass, bot, *args, **dct)[source]¶

Issue a command for a single bot, with optional arguments depending on the command.

CommandClass: must be one of [api.commands.Defend, api.commands.Attack, api.commands.Move]

level = None¶

The LevelInfo object describing the current level.

log = None¶

The logging object that should be used for debug printing.

name = None¶

The name of this commander.

shutdown()[source]¶

Use this function to teardown your bot after the game is over.

tick()[source]¶

Override this function for your own bots. Here you can access all the information in self.game, which includes game information, and self.level which includes information about the level.

You can send commands to your bots using the self.issue() function in this class.

verbose = None¶

Set this to true to enable the display of the bot command descriptions next to each bot.

2.2. Game Info¶

class api.gameinfo.BotInfo(name)[source]¶

Information that you know about each of the bots. Enemy bots will contain information about the last time they were seen. Friendly bots will contain full information.

facingDirection = None¶

The last known movement direction (Vector2) of the bot, None for bots that you have never seen

flag = None¶

The flag being carried by the bot, None if no flag is carried. This is always known for enemy bots.

health = None¶

The current action of a bot (string) (None for unseen enemy bots).

name = None¶

The name of this bot

position = None¶

The last known position (Vector2) of the bot, None for bots that you have never seen.

seenBy = None¶

List of BotInfo objects for enemies which are visible by the team and can see this bot

seenlast = None¶

The time since this bot was last seen, None for bots that you have never seen

state = None¶

The last known state of this bots for enemies, STATE_UNKNOWN for bots that you have never seen. The possible states of an enemy bot: STATE_UNKNOWN, STATE_IDLE, STATE_DEFENDING, STATE_MOVING, STATE_ATTACKING, STATE_CHARGING, STATE_SHOOTING

team = None¶

The TeamInfo for the team that owns this bot

visibleEnemies = None¶

List of BotInfo objects for enemies which are visible to this bot
